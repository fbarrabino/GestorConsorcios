## 📋 Sobre el Proyecto

Este proyecto es una aplicación Full Stack desarrollada con el objetivo principal de profundizar en la **Arquitectura Backend, el Modelado de Bases de Datos Relacionales y la Lógica de Negocio** compleja que requiere la administración financiera de propiedades.

> **Enfoque de Desarrollo:** La prioridad de esta aplicación fue la solidez del servidor, la seguridad de las transacciones y la integridad de los datos en SQL Server. El frontend web integrado cumple un rol estrictamente funcional y minimalista para demostrar la consumición de los endpoints.

## ⚙️ Arquitectura y Funcionalidades Core

* **Lógica de Expensas:** Algoritmos de cálculo automático de expensas ordinarias y extraordinarias, recargos por mora y prorrateos.
* **Gestión de Entidades:** ABM (Alta, Baja, Modificación) relacional de Consorcios, Unidades Funcionales (Departamentos), Propietarios e Inquilinos.
* **Persistencia de Datos:** Diseño de base de datos normalizada en SQL Server, manejada a través de Spring Data JPA.
* **Arquitectura MVC:** Separación estricta de responsabilidades utilizando el patrón Modelo-Vista-Controlador en el ecosistema Spring.

## 🛠️ Stack Tecnológico

* **Backend:** Java, Spring Boot 3
* **Base de Datos:** Microsoft SQL Server
* **Frontend:** HTML5, CSS3, JavaScript (Integración mediante plantillas)
